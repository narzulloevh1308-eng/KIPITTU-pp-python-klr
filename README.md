# Комплекс лабораторных работ (КЛР)

## Дисциплина
Практикум по программированию

## Описание
Данный репозиторий содержит комплекс лабораторных работ (КЛР),
выполненных в рамках дисциплины «Практикум по программированию».
Все работы реализованы на языке Python в среде разработки Visual Studio Code
и оформлены в соответствии с требованиями курса.

Репозиторий используется как единое хранилище лабораторных работ
и является основой для защиты итоговой лабораторной работы №8.

## Структура репозитория

- **LR_01** — Подготовка и настройка среды разработки (VS Code, Python)
- **LR_02** — Циклы, строки, list comprehension
- **LR_03** — Словари, множества, Counter, функции
- **LR_04** — Исключения, контекстные менеджеры, файлы и модули
- **LR_05** — Two Pointers, сортировки, lambda, поиск
- **LR_06** — Рекурсия и алгоритм Sliding Window
- **LR_07** — Unit testing, рефакторинг, генераторы
- **LR_08** — Итоговый мини-проект  
  - `part1` — решение задачи с использованием алгоритмического шаблона  
  - `part2` — мини-проект с генерацией и обработкой синтетических данных

## Используемые технологии
- Python 3.x
- Visual Studio Code
- GitHub
- unittest
- Алгоритмические шаблоны обработки последовательностей
