# Лабораторная работа №7

## Тема
Unit testing, рефакторинг программного кода,
генераторы и итераторы. Ленивые вычисления.

## Цель
Научиться писать модульные тесты с использованием unittest,
рефакторить код и применять генераторы для эффективной обработки данных.

## Задание
1. Реализовать задачу Sliding Window с проверкой входных данных.
2. Написать unit-тесты для проверки корректности работы функции.
3. Реализовать генератор накопительной суммы.

## Бизнес-логика решения
Алгоритм Sliding Window позволяет обновлять результат при каждом сдвиге окна
за константное время, не пересчитывая данные заново.
Генератор накопительной суммы использует аккумулятор и оператор yield,
что позволяет получать значения по мере необходимости.

## Результат
Функции реализованы корректно, покрыты unit-тестами
и соответствуют требованиям задания.
