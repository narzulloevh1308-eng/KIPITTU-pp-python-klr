# Лабораторная работа №2

## Тема
Циклы, вложенные циклы, строки и list comprehensions.

## Цель
Освоить использование циклов for и while, работу со строками,
а также применение list comprehension для обработки данных.

## Задание
1. Найти максимальный элемент списка с использованием цикла for.
2. Найти сумму всех пар элементов без использования вложенного цикла.
3. Создать таблицу преобразований слов строки.

## Бизнес-логика решения
В первой задаче используется линейный проход по списку с сохранением
текущего максимального значения.
Во второй задаче применяется аккумулятор для подсчёта суммы пар элементов
без вложенных циклов.
В третьей задаче используется разбиение строки на слова и list comprehension
для формирования таблицы преобразований.

## Результат
Все задачи успешно реализованы и корректно выполняются.
