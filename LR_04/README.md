# Лабораторная работа №4

## Тема
Исключения, контекстные менеджеры, работа с файлами и модулями в Python.

## Цель
Изучить механизм обработки исключений,
научиться создавать собственные контекстные менеджеры,
а также работать с файлами и модулями.

## Задание
1. Реализовать контекстный менеджер safe_open.
2. Создать модуль для поиска файлов и анализа каталога.
3. Сохранить результаты работы в формате JSON.

## Бизнес-логика решения
Контекстный менеджер safe_open обеспечивает безопасное открытие файла.
Если файл отсутствует, он создаётся автоматически.
Модуль file_search выполняет обход каталога,
поиск файлов по расширению и подсчёт общего размера,
после чего сохраняет результаты в JSON-файл.

## Результат
Программы корректно обрабатывают исключения,
работают с файлами и сохраняют результаты анализа.
