# Лабораторная работа №1

## Тема
Подготовка рабочего места для командной работы на основе IDE Visual Studio Code
для программирования на языке Python и тестирование программ ПЛЗ.

## Цель
Подготовить и настроить среду разработки Visual Studio Code для работы с Python,
а также проверить её работоспособность на простых примерах программ.

## Задание
1. Установить Python версии 3.x.
2. Установить Visual Studio Code.
3. Установить расширение Python от Microsoft.
4. Выбрать интерпретатор Python.
5. Создать рабочий проект и проверить корректность работы среды разработки.

## Бизнес-логика решения
Для проверки корректной работы среды разработки используется простая программа,
выполняющая вывод текста и арифметическую операцию сложения.
Если программа выполняется без ошибок и выводит корректный результат,
значит среда разработки настроена правильно.

## Результат
Среда Visual Studio Code успешно настроена.
Программа корректно выполняется и выводит ожидаемый результат.
