# Лабораторная работа №6
## Тема
Рекурсия: принципы, стек вызовов и примеры.
Алгоритм Sliding Window и анализ последовательностей.

## Цель
Изучить рекурсивные алгоритмы и принципы работы стека вызовов,
а также освоить алгоритм Sliding Window для эффективной обработки данных.

## Задание
1. Рекурсивно «сплющить» вложенный список.
2. Найти самую длинную подстроку без повторяющихся символов
   с использованием алгоритма Sliding Window.

## Бизнес-логика решения
В первой задаче применяется рекурсия: если элемент является списком,
происходит рекурсивный вызов функции для его обработки.
Во второй задаче используется переменное окно Sliding Window,
которое сдвигается по строке без повторного пересчёта данных,
что обеспечивает линейную сложность алгоритма.

## Результат
Задачи реализованы корректно.
Алгоритмы работают эффективно и соответствуют требованиям задания.
