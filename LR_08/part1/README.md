# Лабораторная работа №8 (часть 1)

## Тема
Обработка последовательностей и алгоритмические шаблоны.

## Цель
Закрепить навыки применения алгоритмических шаблонов
(accumulator, sliding window, two pointers)
при решении прикладных задач.

## Задание
Реализовать решение задачи анализа финансовых расходов
с использованием алгоритма Sliding Window.

## Алгоритмический шаблон
Sliding Window (переменное окно).

## Результат
Реализована функция поиска минимального периода,
за который сумма расходов превышает заданный лимит.
