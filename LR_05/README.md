# Лабораторная работа №5

## Тема
Алгоритм Two Pointers, сортировки, lambda-функции и поиск в Python.

## Цель
Освоить алгоритм Two Pointers,
научиться сортировать данные с использованием key и lambda,
а также сравнить линейный и бинарный поиск.

## Задание
1. Найти пару значений с заданной суммой с помощью Two Pointers.
2. Проверить, является ли строка палиндромом.
3. Выполнить сортировку данных по разным критериям.
4. Реализовать поиск элемента.

## Бизнес-логика решения
Алгоритм Two Pointers позволяет находить пары элементов за один проход по данным.
Для сортировки используется встроенная функция sorted с параметром key.
Поиск элемента реализуется линейным способом.

## Результат
Все задачи реализованы корректно и работают с ожидаемой сложностью.
