# Лабораторная работа №3

## Тема
Словари, множества, Counter и функции в языке Python.

## Цель
Освоить использование словарей и множеств,
научиться применять класс Counter для анализа данных,
а также закрепить навыки работы с функциями.

## Задание
1. Определить самые частые типы ошибок в сообщениях Python.
2. Построить рейтинг слов, встречающихся в ошибках.
3. Определить ошибки, характерные для начинающих программистов.

## Бизнес-логика решения
Тип ошибки определяется как первое слово до символа двоеточия.
Для подсчёта частот используется класс Counter.
Для выявления ошибок новичков применяется заранее заданный набор
типичных ошибок начинающих программистов.

## Результат
Программа корректно анализирует сообщения об ошибках
и выводит статистику в удобном для анализа виде.
